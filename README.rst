==============================
ravenos (ra·vuh·nuhs)
==============================

An experimental 64-bit RISC-V operating system kernel written in Rust.

Ravenos is a bare-metal kernel targeting the RISC-V architecture. The goal is to progressively implement the core subsystems of a real operating system (boot, console I/O, batch execution, memory management) on QEMU-emulated RISC-V hardware (real hardware later), without relying on any existing OS or runtime.
